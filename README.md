## `BATTLE LOG / 为什么继续学`

- 年龄会增加机体里程，但不是停机命令。
- 学得慢一点没关系，今天弄懂一点，经验值就是 `+1`。
- 做出来的东西不一定成熟，但应该真实、能运行、能说明白。
- 自己踩过的坑，尽量整理成文档、工具或案例，留给下一位驾驶员。
- 保留一点幽默、好奇心和战斗力。

<details>
<summary><b>为什么叫 ROOKAGE？</b></summary>

`ROOKAGE` 来自 `ROOKIE + AGE`：一个有一点年纪的菜鸟。

头像是一台戴着粗框老花镜的像素机甲。像素感来自那个属于我们的超级任天堂与机器人战略游戏年代；眼镜是一点自嘲，机甲则提醒自己——有年龄，但还有冲劲，还有战斗力。

</details>
